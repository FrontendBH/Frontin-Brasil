CONSTRUINDO O PRIMEIRO SITE (PARTE 4)

Deixando nosso layout mais Agradável

Começamos nosso layout com um reset para retirar o espaçamento do corpo da página, mas antes crie uma pasta chamada css e nela um arquivo chamado main.css após feito isso link sua folha estilo em sua página html caso ainda não saiba é desse jeito abaixo, mas sem os epaços entre o início e fim da tag confira:

< link href="css/main.css" rel="stylesheet" >

Após ter linkado a folha de estilo do seu site, abra o arquivo css e nele escreva os seguintes códigos: 

								/*Reset*/
								html, body, div, span, applet, object, iframe,
							h1, h2, h3, h4, h5, h6, p, blockquote, pre,
							a, abbr, acronym, address, big, cite, code,
							del, dfn, em, img, ins, kbd, q, s, samp,
							small, strike, strong, sub, sup, tt, var,
							b, u, i, center,
							dl, dt, dd, ol, ul, li,
							fieldset, form, label, legend,
							table, caption, tbody, tfoot, thead, tr, th, td,
							article, aside, canvas, details, embed, 
							figure, figcaption, footer, header, hgroup, 
							menu, nav, output, ruby, section, summary,
							time, mark, audio, video {
								margin: 0;
								padding: 0;
								border: 0;								
								font-size: 100%;
								font: inherit;
								vertical-align: baseline;}
							article, aside, details, figcaption, figure, 
							footer, header, hgroup, menu, nav, section {display: block;}
							:root{line-height: 1.4;}
							ol, ul {list-style: none;}
							blockquote, q {quotes: none;}
							blockquote:before, blockquote:after,
							q:before, q:after {
								content: '';
								content: none;
							}
							table {
								border-collapse: collapse;
								border-spacing: 0;
							}

Esse código acima será responsável pelo reset de nosso site, feito isso mande um Ctrl+S na folha de estilo. Prosseguindo em nosso layout temos agora o nosso header onde iremos apresentar nosso site, afinal este será nosso cartão de visitas ;) ! Em sua folha de estilo escreva o seguinte código: 

								/*header*/
								.topo{
									background-image: url(../img/header.jpg);
									background-repeat: no-repeat;
									background-size: 100%;
									width: 100%;
								}
								@media(min-width: 768px){
									.topo{
										background-attachment: fixed;
										height: 80vh;
									}
								}

Se você está iniciando agora deve ter percebido que nosso layout está com alguns códigos diferentes não é mesmo? Afinal o que é essa @media e este background-attachment: fixed;? Esse @media se refere às mídias queryes que servirão para orientar os containers de nosso layout em telas diferentes e até mesmo corrigir algo caso tenha algum dificuldade em se ajustar em alguma tela, caso queira ver a respeito sobre o design responsivo clique aqui 

								/*Overlay*/
								.overlay{
									background-color: #0006;
									height: 100%;
									width: 100%;
								}

Feito isso perceba que corrigimos nosso problema da imagem que poderia interferir no nosso texto. Agora que temos nossa base do header pronto podemos partir para as nossas grids e containers, para acessar a Quinta parte clique aqui.
