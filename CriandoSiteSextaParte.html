CONSTRUINDO O PRIMEIRO SITE (PARTE 6)

Finalizando nosso header

Finalizamos no post anterior nossa grid, agora temos que finalizar nosso header confira abaixo se seu header está assim:

Caso esteja desse jeito está tudo indo ok! Temos uma imagem de fundo em nosso header bem legal, mas falta o nosso texto do header até então será responsável pelas boas vindas ao site. Para isto escreva a tag h1 dentro da tag col-12 e claro digite seu nome dentro da h1. Feito isso então voltaremos ao nosso CSS onde iremos incrementar os seguintes códigos no nosso header: 

								.topo .container{padding-top: 10em;}
								.topo h1{
									color: #fff;
									font-size: 3.6em;
									text-align: center;
								}
								@media(min-width: 768px){
									.topo{
										background-attachment: fixed;
										height: 80vh;
									}
									.topo h1{font-size: 6em;}
								}

Você pode observar que nosso header está ganhando vida aos poucos não é mesmo? Só que ao observarmos nosso header vemos que nossa fonte está com uma cara muito feia, então que tal se a gente colocar uma mais clean?! Para usaremos o Google Fonts um site da empresa Google que disponibiliza diversas fontes de livre uso, para acessar o link do site clique aqui,como estamos apenas fazendo um exemplo usarei na fonte do header a Bungee Inline no em tanto ao fazer seu layout procure usar fontes cleans que não venha a poluir o seu site e que também não sejam ruins de leitura. Outro detalhe importante caso use o link import do css, sempre coloque ele no topo de seu site para que o carregamento do link seja o primeiro. Agora para finalizar de vez nosso header escreva a tag p dentro da coluna 12 de seu header e nela escreva uma frase legal de boas vindas, eu usarei a seguinte frase "Olá sejam bem-vindos ao meu portfa!". Assim que escrever sua frase insira o seguinte código CSS: 

								.topo p{
									color: #fff;
									font-family: 'Raleway', sans-serif;
									font-size: 2em;
									text-align: center;
								}
